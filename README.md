# 🎮 RPG Student Skill Tree Simulator

> An AI-Powered gamified learning platform that combines RPG game mechanics, interactive skill trees, personalized AI mentoring, Vibe Coding, and a rich community hub — all running 100% in the browser with no backend required.

![Skill Tree Simulator](https://img.shields.io/badge/Status-Live-brightgreen) ![Tech](https://img.shields.io/badge/Built%20With-HTML%2FCSS%2FJS-cyan) ![License](https://img.shields.io/badge/License-MIT-violet)

---

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/incharassathish77-dotcom/acp_practice_cls_1.git
cd acp_practice_cls_1

# Serve locally (requires Node.js)
npx http-server -p 8080

# Then open:
# http://localhost:8080
```

No build step. No dependencies to install. Just open and play.

## 🌟 Features

| Feature | Description |
|---|---|
| 🌳 **RPG Skill Tree** | 15 interactive nodes across 6 specialization tracks with SVG connection lines |
| 🧠 **AI Mentor Chat** | Simulated AI tutor with keyword-routing, study plan generation, and glossary lookups |
| 🎯 **Career Advisor** | Real-time skill gap analysis for 6 tech careers (AI Engineer, Data Scientist, etc.) |
| 🎵 **Vibe Coding Hub** | Prompt → real-time AI code typing simulation with file trees and milestone tracking |
| 📖 **AI Glossary** | 12 searchable AI/ML terms with beginner-friendly analogies and related concept tags |
| 🛠️ **Tools Explorer** | 12 modern AI tools (ChatGPT, Cursor, Midjourney, etc.) with comparisons and categories |
| 👥 **Community Lounge** | Dynamic leaderboard, social feed, and joinable study guilds |
| 📊 **Analytics Dashboard** | Radar chart tracking skill mastery, XP, streaks, and daily missions |
| 🎉 **Gamification** | XP system, level-ups with confetti, streak tracking, badges, and daily missions |
| 🌙 **Dark/Light Mode** | Full theme switching with persistent preference |
| 💾 **Data Backup** | JSON export/import for cross-device profile sync |

## 🗂️ Project Structure

```
acp_practice_cls_1/
├── index.html        # Main SPA shell — all views, sidebar HUD, modals
├── style.css         # Design system — cyberpunk dark/light themes, animations
├── app.js            # State manager, router, game loops, view renderers
├── aiEngine.js       # Simulated AI: Mentor, Career Advisor, Challenge Generator, Vibe Coder
├── skillsData.js     # 15 skill nodes with lessons, quizzes, coding challenges
├── careerData.js     # 6 career paths with roadmaps and requirements
├── toolsData.js      # 12 AI tools across 5 categories
├── glossaryData.js   # 12 AI/ML terms with definitions and analogies
└── communityData.js  # Mock leaderboard, social feed, study guilds
```
